Projetos em Destaque

Marcador de Truco com ATmega328P

Desafio: Desenvolver um placar digital e portátil para jogos de truco, substituindo a marcação manual. O principal desafio técnico foi implementar a lógica de multiplexação de display (time-division multiplexing) para controlar um display de 7 segmentos duplo diretamente com o ATmega328P, utilizando transistores para a comutação dos dígitos.

Tecnologias: Microcontrolador ATmega328P, Programação C++ (Arduino IDE), Display de 7 Segmentos (Duplo), Transistores (para multiplexação), Componentes Eletrônicos (capacitores, resistores).

Smart Liquid

Desafio: Projetar um sistema de monitoramento via Internet (IoT). O desafio foi integrar múltiplos sensores (temperatura e nível de água) e um atuador (um servomotor simulando uma fechadura) em um ESP32. O sistema deveria coletar dados em tempo real e enviá-los via Wi-Fi para uma plataforma ou site, permitindo monitoramento remoto.

Tecnologias: ESP32 (com Wi-Fi), Sensor de Temperatura (DS18B20), Sensor de Água(HW), Servomotor (SG90), Programação C++ (Arduino IDE), Progamação HTML, CSS, JavaScript, PHP e Mysql para o site.

Cuba Automatizada

Desafio: Desenvolver uma 'cuba de pia' totalmente automatizada e *touchless* (sem contato), focando em higiene e eficiência. O projeto exigiu a integração de três sistemas independentes: 1) Um sensor de presença (PIR) para acionamento automático da iluminação. 2) Um sensor infravermelho para detectar as mãos e acionar a bomba d'água da torneira. 3) Um segundo sensor infravermelho para acionar o secador de mãos.

Tecnologias:Sensor de Presença (PIR), Sensores Infravermelhos (x2, E3F-DS30C4), Relé (para controle da bomba e secador), Bomba d'água.
